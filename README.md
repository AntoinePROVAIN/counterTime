# CounterTime

Le projet à pour but de permettre de faire le décompte des heures que ce soit à la fois par mois ou par années. 

## Fonctionnement

Une fois sur le lien du site, cela vous demande une authentification. Pour le moment, rentrez uniquement votre nom et faites connexion. Cette partie là n'est pas encore fonctionnelle. 
Lorsque vous avez fait cela, vous vous retrouvez sur la page centrale, avec l'affichage pour l'année des jours de travail. En fonction de cela, vous avez la possibilité pour chaque jour de n'importe quel mois de rentrer le nombre d'heures travaillées. Cela vous fera la somme totale en bas de celle-ci, à la fois par mois et par an.

## Amélioration à venir

Il faudrait faire en sorte de connecter cela à une base de données ou à un fichier json permettant de stocker les valeurs rentrées afin de faire le suivi sur une année.
